# Veguitas API

API to manage veguitas' website and dashboard

## Tech Stack
* JavaScript
* TypeScript
* Node
  * Express
  * Multer
* MongoDB
* Socket IO
* Amazon Web Services

## Features
* User management
* Songs requests
* Reservations

## Setup

The project is bundle split in 3 sub applications: 
* Backend API
* Dashboard
* Website

After cloning the project, you will need to run `npm install` to install all dependencies

### To run the API:
`npm run dev`
